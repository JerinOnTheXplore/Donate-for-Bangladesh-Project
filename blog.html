What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree-like structure. Each element, attribute, and text within the document is represented as a node in this hierarchical model. The DOM allows developers to interact with and manipulate the content and structure of a webpage dynamically using JavaScript.

How do you select an element from the DOM?

There are several methods to select elements from the DOM using JavaScript:
1. document.getElementById("id") – Selects an element by its unique ID.
2. document.getElementsByClassName("class") – Selects elements by class name (returns a collection).
3. document.getElementsByTagName("tag") – Selects elements by tag name (returns a collection).
4. document.querySelector("selector") – Selects the first matching element using a CSS selector.
5. document.querySelectorAll("selector") – Selects all matching elements using a CSS selector (returns a NodeList).

Why is event delegation useful?

1. Efficient event handling: Reduces the number of event listeners needed, improving performance.
2. Handles dynamically added elements: Useful when elements are added or removed dynamically.
3. Simplifies code: Keeps event handling logic centralized and easier to maintain

How do you manipulate an element's attributes and styles using the DOM?

Changing attributes:
1. element.setAttribute("attribute", "value") – Sets an attribute.
2. element.getAttribute("attribute") – Gets an attribute.
3. element.removeAttribute("attribute") – Removes an attribute.
4. element.style.property = "value" – Modifies an inline CSS property.
5. element.classList.add("class") – Adds a CSS class.
6. element.classList.remove("class") – Removes a CSS class.
7. element.classList.toggle("class") – Toggles a class on or off.
